## Colour usage

| Purpose | Class | Token |
|---------|-------|-------|
| Primary text | `text-foreground` | `--foreground` |
| Secondary text | `text-muted-foreground` | `--muted-foreground` |
| Destructive text | `text-destructive` | `--destructive` |

## Usage guidelines

- **H1 / H2** — page titles and major section headers only.
- **H3 / H4** — card titles, modal headings, sidebar sections.
- **Body base** — default running text.
- **Body small** — metadata, captions, helper text.
- **Monospace** — code blocks, IDs, hashes. Apply `font-mono`.
