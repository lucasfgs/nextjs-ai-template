## Composition guidance

<DoDont
  doTitle="Compose from shared building blocks"
  doBody={
    <>
      Start with `Card`, `Label`, `Input`, `Alert`, `Badge`, and `Button`, then layer layout tokens
      like `gap-stack` and `shadow-elevation-sm` to create predictable product surfaces.
    </>
  }
  dontTitle="Invent a new layout grammar for every screen"
  dontBody={
    <>
      Avoid copy-pasting one-off wrappers with hard-coded colors, spacing, or shadow values. That
      makes theming harder and Storybook less useful as a reference.
    </>
  }
/>
