## Review checklist

- Tab through every interactive state in Storybook.
- Test light and dark themes for visible focus, contrast, and disabled-state clarity.
- Keep descriptive copy close to fields, destructive actions, and empty states.
- Prefer semantic HTML and shared Radix primitives before reaching for custom interaction logic.
