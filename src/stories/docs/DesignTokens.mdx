# Design Tokens

All tokens are CSS custom properties defined in `src/app/globals.css`. They are mapped to Tailwind v4 utilities via `@theme inline {}`.

## Colour palette

<div style={{ display: 'grid', gridTemplateColumns: 'repeat(auto-fill, minmax(200px, 1fr))', gap: '16px', margin: '24px 0' }}>
  {[
    ['Background', 'background', 'foreground'],

      </div>
    </div>
  ))}
</div>

## Border & Focus

| Token | CSS Variable | Tailwind class |
|-------|-------------|----------------|
| Border | `--border` | `border-border` |
| Input | `--input` | `border-input` |
| Focus ring | `--ring` | `ring-ring` |

## Border radius

| Scale | CSS Variable | Value |
|-------|-------------|-------|
| `sm` | `--radius-sm` | `calc(var(--radius) - 4px)` |
| `md` | `--radius-md` | `calc(var(--radius) - 2px)` |
| `lg` | `--radius-lg` | `var(--radius)` = `0.5rem` |
| `xl` | `--radius-xl` | `calc(var(--radius) + 4px)` |

## Typography

| Property | CSS Variable | Value |
|----------|-------------|-------|
| Sans font | `--font-sans` → `--font-geist-sans` | Geist |
| Mono font | `--font-mono` → `--font-geist-mono` | Geist Mono |

## Customising tokens

Override any token inside a scoped CSS selector to white-label a section:

```css
/* Tenant-specific overrides */
.brand-acme {
  --primary: #1d4ed8;
  --primary-foreground: #ffffff;
}
```

All components that use `bg-primary` / `text-primary-foreground` will automatically reflect the change.
