## Form and chrome tokens

| Purpose     | CSS variable | Tailwind utility        | Guidance                                                      |
| ----------- | ------------ | ----------------------- | ------------------------------------------------------------- |
| Borders     | `--border`   | `border-border`         | Default outlines for surfaces, separators, and field chrome.  |
| Inputs      | `--input`    | `border-input`          | Default form field edge treatment.                            |
| Focus ring  | `--ring`     | `ring-ring`             | Use with visible focus styles on every interactive primitive. |
| Radius base | `--radius`   | via `rounded-*` aliases | Shared shape contract for all primitives.                     |

## Branding override pattern

```css
.brand-acme {
  --primary: #1d4ed8;
  --primary-foreground: #ffffff;
  --ring: #1d4ed8;
}
```

Scope overrides to a wrapper when you need tenant or campaign-specific branding. Because the
components consume semantic tokens instead of raw color values, the UI will inherit the new brand
without rewriting button, badge, input, or dialog classes.
